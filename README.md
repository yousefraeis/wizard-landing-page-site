## Getting Started

To run this project locally, follow these steps:

1. Clone the repository:
   ```bash
   git clone https://github.com/yousefraeis/wizard-landing-page-site.git
   ```
2. Navigate to the project directory:
   ```bash
   cd wizard-landing-page-site
   ```
3. Install the dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```

You can view the project in your browser at `http://localhost:3000`.

## Project Structure

- `pages/`: Contains the main application pages.
- `components/`: Reusable UI components for easy customization.
- `styles/`: Global styles and Tailwind CSS configuration.
- `public/`: Static assets such as images and icons.
